## MiniDB, a tiny implementation of bitcask.
使用 Java 对 Bitcask KV存储模型的简单实现。

#### 已完成内容
+ 多数据文件
  - 数据文件名使用整数命名，递增
  - 单文件限制最大大小
+ hint 文件
  - 为同名数据文件的 hint 文件
+ 数据文件压缩
  - 单线程定时任务
  - 压缩阈值

#### 文件目录示例
+ 以名为 test 的数据库实例为例：
    
    ```text
    .
    └── test
        ├── data
        │    ├── 19
        │    ├── 20
        │    ├── 21
        │    ├── 22
        │    ├── 23
        │    └── 24
        ├── hint
        │    ├── 19
        │    ├── 20
        │    ├── 21
        │    ├── 22
        │    └── 23
        └── meta
    ```

+ meta 存储文件记录了一些元数据：最新的和活跃文件，最近一次数据文件压缩时的活跃数据文件和压缩进度。
+ lock 文件锁

#### 对象序列化
+ 使用Java自带序列化机制。
